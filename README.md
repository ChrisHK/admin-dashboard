# Admin Dashboard System

A full-stack admin dashboard system built with React, Node.js, and PostgreSQL.

## Features

- User Authentication
- User Management
- Product Management
- Responsive Design
- Docker Support

## Tech Stack

- Frontend:
  - React
  - Material-UI
  - React Router
  - Axios

- Backend:
  - Node.js
  - Express
  - PostgreSQL
  - JWT Authentication
